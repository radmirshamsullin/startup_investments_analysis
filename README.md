# Исследование стартапов

**Цель проекта:**  
Проанализировать динамику и структуру инвестиций в стартапы, выявить популярные сегменты рынка и дать рекомендации для инвестиционной стратегии.

**Описание проекта:**  
- Проведена предобработка данных (удаление дубликатов, обработка пропусков, проверка аномалий)  
- Разделены компании на группы по срокам финансирования и сегментам рынка (массовые, средние, нишевые)  
- Проведен анализ объёмов инвестиций по типам финансирования и по годам  
- Рассчитаны доли возврата средств по типам финансирования  
- Построены графики и сводные таблицы для визуализации динамики и закономерностей

**Навыки и технологии:**  
- Python (pandas, numpy, matplotlib, seaborn)  
- Анализ данных и визуализация  
- Работа с большими таблицами и агрегатами  
- Подготовка отчётов и выводов для бизнеса

**Структура репозитория:**
```

├── README.md # текущий файл

```
